    As the comments in this file indicate, these are just definitions for setting and testing bits in bitmap.c. In the 2.6.32 version of the xiafs port it was able to use the minix versions of these, but those went away somewhere between 2.6.32 and 3.2. N.B. This remains unchanged since the annotations for the 3.12.1 kernel version of the xiafs module.
